Utils Class

class Utils.utils

A utility class with static methods for file and geospatial data operations.

Note

All methods in this class are static and do not require class instantiation.

static ID_Index(gdM)

Adds a unique integer index and an ‘ID’ column to a GeoDataFrame.

Parameters:

gdM (geopandas.GeoDataFrame) – Input GeoDataFrame.

Returns:

Updated GeoDataFrame with index and ‘ID’.

Return type:

geopandas.GeoDataFrame

static get_download_urls(url)

Finds downloadable file links from the given URL.

Parameters:

url (str) – The target webpage URL.

Returns:

A list of full URLs pointing to downloadable files.

Return type:

list[str]

static mk_dir(cwd)

Creates a directory if it does not exist.

Parameters:

cwd (str) – The path to the directory.
